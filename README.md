# 🚔 교통외근 보조 앱 (v8.4 Universal)

**서초경찰서 교통외근 실무자를 위한 통합 PWA**  
단일 HTML 파일로 오프라인에서도 완벽 동작. 현장 즉시 사용 가능.

![App Preview](https://via.placeholder.com/800x400?text=App+Screenshot+1)  
*(실제 스크린샷 2~3장 나중에 추가 추천)*

## ✨ 주요 기능

- **📍 관할 자동 판정** — GPS로 서초서 관할 구간 + 통보처 자동 추천
- **🚨 원스톱 사고 보고서** — 음성입력 → 1차(카톡) / 2차(KICS) / 3차(언론) 보고서 일괄 생성
- **🛣️ 자동차전용도로 특화 SOP** — 경부·올림픽대로 서울구간 (시설공단 단독 통보 등 최신 정정사항 반영)
- **⚖️ 법령·판례·핑계대응·심리방어 DB**
- **📞 긴급 연락처 저장** — 시설공단, 119, 서초서 직통 등
- **🎙️ 음성 텍스트 변환** — 안드로이드/아이폰 모두 지원
- **PWA 완벽 지원** — 홈화면 추가 후 앱처럼 사용 (오프라인 OK)

## 📲 설치 방법 (PWA)

1. Chrome / Edge / Safari로 [https://340patrolman.github.io/traffic-app](https://340patrolman.github.io/traffic-app) 접속
2. 주소창 오른쪽 **설치** 버튼 클릭 (또는 메뉴 → "홈 화면에 추가")
3. 홈화면에 앱 아이콘 생성 → **오프라인에서도 사용 가능**

**직접 파일 사용 시**: `index.html` 하나만 다운로드해서 브라우저에서 열기.

## 🛠️ 사용 팁

- **GPS 관할판정**: 현장 도착 즉시 사용
- **음성입력**: 키보드 마이크 또는 앱 내 버튼 활용
- **데이터 저장**: localStorage 사용 (브라우저 클리어 시 초기화됨)
